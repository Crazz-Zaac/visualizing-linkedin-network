### Instructions 
- Download your linkedin data
- Extract and place these files `Company Follows.csv, connections.csv, Connections.csv, multiTimeline.csv` inside the `resources/dataset/` folder

---

### How to run the project?
- `git clone ` this repo
- Create a python environment
- Install the required packages: `pip install -r requirements.txt`
- Run the project: `streamlit run app.py`
